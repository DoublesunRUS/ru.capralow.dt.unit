Имя метода: ПроверитьИстину

Синтаксис:

ПроверитьИстину(_Истина, ДопСообщениеОшибки = "")

Описание:

Проверяет, что переданное значение истинно.

Тип возвращаемого значения: 

Описание возвращаемого значения: 

Параметры:
  • _Истина  (Обязательный).

Тип: Булево.

Описание: Булево.

  • ДопСообщениеОшибки

Тип: Строка.

Описание: Дополнительное сообщение для ошибки.

Доступность:

  • Сервер
  • Толстый клиент (управляемое приложение)
  • Толстый клиент (обычное приложение)

Пример:

ПроверитьИстину(ПроверяемоеЗначение, "Ожидали истину, а получили ложь.")
